# 설계핏 AI MVP v1.2

주니어 설계자를 위한 제조 관점 도면 셀프체크 웹앱 초안입니다.

v1.2는 모바일 앱이 아니라 **PC/CAD 작업 환경 우선**으로 정리한 버전입니다. CATIA, NX, SolidWorks 등 CAD 프로그램 옆에 브라우저를 띄워두고 도면 조건을 빠르게 점검하는 사용 흐름을 기준으로 합니다.

## MVP 범위

- 설계 조건 입력: 부품 유형, 재질, 공정, 검사 방식, 공차, GD&T, 설계 고민
- DFM 리스크 분석: 제조 난이도, 검사 리스크, ECO 가능성, 종합 위험도
- 공차 적정성 가이드: 과도한 공차 여부와 완화 검토 방향
- GD&T 제조 관점 해석: 위치도, 원통도, 평면도, Datum 질문
- 협의 문구 생성: 설계팀/생산기술팀 협의용 문구 복사
- 샘플 분석: 샤프트, 하우징, 용접품, 항공 부품
- 결과 저장: localStorage 기반 최근 검토 내역
- PDF 저장: 브라우저 인쇄 기능 사용

## v1.2 반영 사항

- 모바일 중심 반응형 UX보다 PC 워크벤치 구조 우선
- CAD 도면 검토 문맥으로 상단 카피와 입력 문구 수정
- 넓은 화면에서 좌측 입력, 우측 결과가 안정적으로 보이는 2열 구조 강화
- 결과 패널 sticky 처리로 입력 스크롤 중에도 분석 결과 확인 가능
- "PC / CAD 작업 환경 권장" 표시 추가

## 로컬 실행

```bash
node server.js
```

브라우저에서 `http://localhost:4173`을 엽니다.

## Vercel 배포

정적 웹앱이므로 GitHub 저장소를 Vercel에 연결하면 바로 배포할 수 있습니다.

- Framework Preset: Other
- Build Command: 비워두기
- Output Directory: `.`
