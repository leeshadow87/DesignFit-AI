# DesignFit-AI V5

DesignFit-AI는 CAD/제조 도면을 읽고 공차, GD&T, 재질, 공정 난이도, 검사 방식, 비용 영향, CAE 검증 필요성을 함께 정리하는 제조성 AI 검토 콘솔입니다.

## 제품 방향

비숙련 설계자는 기능 요구보다 과도하게 타이트한 공차를 지정하는 경우가 많습니다. 이로 인해 가공 비용 증가, 특수 공정 필요, 검사 난이도 증가, 납기 지연, 협력사 선택 폭 축소가 발생합니다.

V5의 첫 번째 핵심 능력은 공차 완화 검토입니다. 다만 제품은 단순한 공차 계산기가 아니라 설계자가 도면을 제출하기 전에 제조 관점의 위험을 빠르게 확인하는 AI 검토 콘솔을 목표로 합니다.

## V5 MVP 범위

- PDF 도면 업로드 및 뷰어
- PDF text layer 기반 공차/GD&T 후보 추출
- 수동 보정 입력
- 공차 위험도 분류
- 완화 후보 및 검증 질문 제안
- 제조성 리스크 요약
- 공개 샘플 도면 기반 시제품 검증

## 보안 주의

현재 버전은 검증용 시제품입니다.

- 실제 고객 도면 업로드 금지
- 사내 도번, 고객명, 업체명, 프로젝트명 노출 금지
- 공개 도면 또는 합성 샘플 도면만 사용 권장
- localStorage 기반 저장이므로 운영용 보안 저장소가 아닙니다.

운영 전에는 인증, 비공개 파일 저장소, 접근 권한, 파일 삭제 정책, 로그 관리, 고객별 데이터 격리가 필요합니다.

## 실행

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:3000`으로 접속합니다.
